CAPACITANCE MEASUREMENT CIRCUIT - KICAD REFERENCE
=================================================

SCHEMATIC:

         D7 ────────┬────────────────────────────────┐
                    │                                │
                    │    ┌───────┐                   │
                    └────┤ 2.2MΩ ├───────┬───────────┼───── TIP (to cable)
                         └───────┘       │           │
                              R1         │           │
                                         │           │
         D8 ─────────────────────────────┤           │
                    │                    │           │
                    │    ┌───────┐       │           │
                    └────┤  1kΩ  ├───────┘           │
                         └───────┘                   │
                              R2                     │
                                                     │
         A1 ─────────────────────────────────────────┘
                              │
                         (sense node)


COMPONENTS:
  R1: 2.2MΩ  1/4W  1% metal film
  R2: 1kΩ    1/4W  5% OK

NETS FOR KICAD:
  CAP_CHARGE    : D7 → R1 pin 1
  CAP_NODE      : R1 pin 2 + R2 pin 2 + A1 + TIP (all joined)
  CAP_DISCHARGE : D8 → R2 pin 1
  GND           : SLEEVE (via K1+K2 relay)

CONNECTION POINT:
  The CAP_NODE connects to TIP at the near-end test jack.
  SLEEVE goes to GND when K1+K2 relay is LOW (shorts far end).
